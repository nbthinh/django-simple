# Setup

- python3 -m venv venv
- source venv/bin/activate
- django-admin startproject djangoapi

## Khi khởi tạo project

- pip3 install django
- pip3 install djangorestframework

## Với project có sẳn

- Clone về
- cd vào project
- pip3 install -r requirements_example.txt

## Startapp

- python3 manage.py startapp courses
